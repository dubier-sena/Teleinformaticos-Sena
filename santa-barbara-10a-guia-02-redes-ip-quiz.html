Quiz individual de parámetros de red

Presenta un quiz aleatorio de 10 preguntas sobre estructura de IPv4, clases de IP, direcciones privadas, máscara de subred, gateway y DNS. El intento usa Page Visibility API y se finaliza automáticamente en la segunda salida de pestaña.

Reglas del intento

  • El quiz usa una variante al azar entre A, B y C al momento de iniciar.
  • Son 10 preguntas de selección múltiple con puntaje máximo de 100.
  • Hay 2 advertencias por cambio de pestaña. En la segunda advertencia el sistema envía lo contestado y bloquea el intento.
  • Cuando el intento queda finalizado, ya no se permite contestar más.
Volver a la guia
Listo para iniciar. La variante se asignará al azar cuando pulses "Iniciar quiz".

Cuestionario

Pulsa Iniciar quiz para cargar la variante aleatoria y comenzar el intento. Esta página mantiene las preguntas fuera del HTML estático y las renderiza dinámicamente.
